# Termodinamična analiza in izračun potreb po energiji za ogrevanje objekta

## Opis projekta:

V tem projektu bom analiziral potrebo po toplotni energiji za ogrevanje stanovanjskega objekta v določenem časovnem obdobju. Podatke o urnih zunanjih temperaturah bom pridobil s pomočjo vgrajenih funkcij. Analiziral bom gibanje zunanje temperature skozi čas in ugotavljal, kdaj nastopi temperaturni primanjkljaj.
